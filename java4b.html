Java - Desarrollo e IDEs

Compilar utilizando interfaz de comandos

Para desarrollar en Java, solamente es necesario tener un JDK (Java Development Kit) compatible con el sistema operativo que se desea usar.

Al instalar el JDK obtenemos acceso a la interfaz de comandos de utilidad para desarrollo en Java. Esto incluye el comando javac (abreviación de "Java compile"), el cual podemos utilizar para compilar un archivo Java (por ejemplo, Main.java:

javac Main.java

Esto genera un archivo .class que contiene el bytecode listo para ejectura de nuestro programa. Luego para ejecutar nuestro programa basta con ejecutar:

java Main

Consideremos que nuestro archivo Main.java contiene lo siguiente:

public class Main { public static void main(String[] arguments) { System.out.println("Prueba de compilado"); } }

Al correr java Main obtendríamos el siguiente resultado en la consola:

Prueba de compilado

IDEs Recomendados

  • IntelliJ Idea: Considerado el estandar para desarrollo en Java profesional. Incluye soporte para tecnologías web.
  • Eclipse: Existen dos versiones principales, una para desarrollo en Java puro solamente, y otra (Eclipse EE) con soporte para tecnologías web y ciertas bases de datos.
