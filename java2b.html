Java - Historia y Versiones

Java fue originalmente desarrollado por James Gosling, y fue publicado por Sun Microsystems en 1995.

Se creo con el propósito de ser un lenguaje orientado a objetos, basado en clases, con tipado fuerte, y en particular, diseñado para ser lo más independiente de plataforma como fuera posible. Para lograr esto, el código escrito en Java pasa por un proceso de "pre-compilación" donde se prepara el código para ser ejecutado no como lenguaje de máquina, sino como instrucciones para una máquina virtual de Java. Esta máquina virtual se encarga de librar al código de la responsabilidad de lidiar con dependencia de alguna plataforma específica.

Versiones notorias

  • Java 8: Una de las versiones más importantes. Introdujo múltiples cambios y reemplazos a APIs antiguos, y también introdujo soporte parcial para programación funcional, lo cual permitio hasta cierto punto reducir el uso excesivo de clases y objetos. Los cambios de lenguaje introducidos por Java 8 fueron tan importantes que algunas compañías lo consideran un lenguaje independiente de versiones anteriores.
  • Java 9: Introdujo el sistema de módulos nuevo, junto con cambios y adiciones pequeñas a la librería estandar.
  • Java 11: Es la última versión, hasta el día actual, con soporte a largo plazo.
  • Java 14: La última versión, publicada el marzo del 2020.
